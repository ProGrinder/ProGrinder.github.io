基础算法-排序

快速排序

快排基于分治思想

Step1: 确定分界点: 可选arr[l], arr[(l + r)/ 2] , arr[r];

Step2: 调整区间，使得：左区间的数≤分界点 ， 右区间的数≥分界点; ⭐️

Step3: 递归处理左右两个区间。

复杂度分析

时间复杂度O(nlogn)O(nlogn)O(nlogn) 空间复杂度 不稳定

算法实现

Step2双指针实现：

第一个指针最初指向最左端， 第二个指针最初指向最右端。

将左右两个指针不断往中间移动移动，

当左边的指针第一次遇到>边界值的数，右边的指针第一次遇到<边界值的数，交换指针指向两个元素。

直到两指针相遇，完成当轮次快排。

可以发现这个算法有许多边界问题需要注意，因此还是建议背模板的!

快速排序模板:

#include <vector>
#include <iostream>

using namespace std;

void quick_sort(vector<int> &arr, int l, int r) {
	// 递归边界：区间里没有数了或者只有一个数
	if (l >= r) return;

	int x = arr[l + (r - l) / 2];
    
	int i = l - 1, j = r + 1;
	while (i < j) {
		do i ++; while (arr[i] < x);
		do j --; while (arr[j] > x);
		if (i < j) swap(arr[i], arr[j]);
	}
    
	quick_sort(arr, l , j);
	quick_sort(arr, j + 1, r); 
}

int main() {
	int n;
	cin >> n;

	vector<int>arr(n);
	for (int i = 0; i < n; i ++) scanf("%d", &arr[i]);

	quick_sort(arr, 0, n - 1);

	for (int i = 0; i < n; i ++) printf("%d ", arr[i]);

	return 0;
}

细节1： 必须要同时使用 i - 1, i 或 j , j + 1

当使用i - 1, i， x不能取左边界[l]， 当使用j, j + 1时, x不能取右边界arr[r]
否则就会出现边界问题，陷入无限递归死循环

细节2: l + (r - l) / 2和 l + (r - l + 1) / 2

C++的/是向下取整的，因此l + (r - l) / 2偏向左边界[l] =>用j , j + 1;
人为+1/2进行四舍五入，因此l + (r - l + 1) / 2偏向右边界[r] => 用 i - 1, i

细节3: while (arr[i] < x); while (arr[j] > x);就连等于边界值x时都要交换

细节4: int i = l - 1, j = r + 1;初始化时i, j在两边界外

归并排序

归并排序基于分治思想

Step1: 以中间点为分界点: mid = l + (r - l) / 2;

Step2: 递归排序left, right区间;

Step3: 归并—把两个有序子数组合二为一 。⭐️

复杂度分析

时间复杂度O(nlogn)O(nlogn)O(nlogn) (妥妥的) 稳定

算法实现

Step3双指针实现：

第一个指针最初指向第一个有序子数组(从小到大)的开头，min1min_1min1​

第二个指针最初指向第一个有序子数组(从小到大)的开头，min2min_2min2​

对两个指针指向的数循环比较，将较小的数归并回原数组，然后其指针右移，继续比较；

直到两个指针都指向其数组的末端，完成当轮此归并。

归并排序模板：

#include <vector>
#include <iostream>

using namespace std;

vector<int> tmp;//要开辟一个数组做归并时的临时存储

void merge_sort(vector<int> &arr, int l, int r) {
    // 递归边界
	if (l >= r) return;
    
    int mid = l + (r - l) / 2;
    
    merge_sort(arr, l, mid);
    merge_sort(, mid + 1, r);
    
    int k = 0;// tmp数组已经归并的元素数量
    int i = l, j = mid + 1;
    
    while (i <= mid && j <= r) {
        if (arr[i] <= arr[j]) tmp[k ++ ] = arr[i ++];
        else tmp[k ++] = arr[j ++];
    }
    //复制末尾
    while (i <= mid) tmp[k ++] = arr[i ++];
    while (j <= r) tmp[k ++] = arr[j ++];
    //复制回原数组进行覆盖
    for (int i = l, j = 0 ; j < k; i ++, j ++) arr[i] = tmp[j];
}

int main() {
	int n;
	cin >> n;

	vector<int>arr(n);
    tmp.resize(n);
    
	for (int i = 0; i < n; i ++) scanf("%d", &arr[i]);

	merge_sort(arr, 0, n - 1);

	for (int i = 0; i < n; i ++) printf("%d ", arr[i]);

	return 0;
}
