树状数组

树状数组（Binary Index Tree / Fenwick Tree）是一种简单的数据结构，主要用来维护序列的前缀和。
它根据任意正整数关于2的不重复次幂的唯一分解性质，把区间二进制分解，支持在O(log⁡2n)O(\log_2n)O(log2​n)对原序列单点更新，同时支持在O(log⁡2n)O(\log_2 n)O(log2​n)内查询前缀和。
