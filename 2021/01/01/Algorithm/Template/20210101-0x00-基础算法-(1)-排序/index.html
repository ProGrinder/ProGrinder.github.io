0x00-基础算法-(1)-排序

排序

排序算法有很多，算法题中最常用就以下几种：

std::sort

sort()函数(Defined in header <algorithm>)

这个算法是个缝合怪，它汲取了插入排序、堆排序以及快排的优点，通常可以看作快排。

一般都结合Lambda表达式，写一个匿名的比较函数使用:

#include <vector>
#include <algorithm>

struct Node {
    int value; // 价值
    int cost;  // 花费
}

vector<Node> arr;
// 按(value / cost)排序
std::sort(arr.begin(), arr.end(), [](const Node& a, const Node& b) -> bool {
    // a.value / a.cost < b.value / b.cost，把除法改成乘法，避免精度问题
    return a.value * b.cost < b.value * a.cost;
});

如果要体现命名规范，a 可以写成 lhs, b 可以写成rhs

复杂度分析

  • 时间复杂度：O(nlog⁡n){O(n\log n)}O(nlogn)，通常跟快排的时间复杂度相同。

快速排序

快排基于分治思想

Step1: 确定分界点: 一般取中点x = arr[l + (r - l) / 2](这个分界点会很大程度上决定时间复杂度的好坏)

Step2: 调整区间，使得：左区间的数≤分界点 ， 右区间的数≥分界点; ⭐️

Step3: 递归处理左右两个区间。

算法实现

实现难点主要在于Step2双指针实现：

  • 第一个指针最初指向最左端， 第二个指针最初指向最右端。

  • 将左右两个指针不断往中间移动移动，
    当左边的指针第一次遇到>边界值的数，右边的指针第一次遇到<边界值的数，交换指针指向两个元素。
    直到两指针相遇，完成当轮次快排。

快速排序模板:

#include <vector>
#include <iostream>

using namespace std;

void quick_sort(vector<int> &arr, int l, int r) {
	// 递归边界：区间里没有数了或者只有一个数
	if (l >= r) return;

    // step1
	int x = arr[l + (r - l) / 2];
    
    // step2
	int i = l - 1, j = r + 1;
	while (i < j) {
		do i ++; while (arr[i] < x);
		do j --; while (arr[j] > x);
		if (i < j) swap(arr[i], arr[j]);
	}
    
    // step3
	quick_sort(arr, l , j);
	quick_sort(arr, j + 1, r); 
}

int main() {
	int n;
	cin >> n;

	vector<int>arr(n);
	for (int i = 0; i < n; i ++) scanf("%d", &arr[i]);

	quick_sort(arr, 0, n - 1);

	for (int i = 0; i < n; i ++) printf("%d ", arr[i]);

	return 0;
}

复杂度分析

  • 时间复杂度：平均O(nlog⁡n)O(n\log n)O(nlogn)，最坏时间复杂度O(n2)O(n^2)O(n2)

  • 空间复杂度：O(log⁡n){O(\log n)}O(logn) ，递归深度log⁡n{\log n}logn，即函数栈开销log⁡n{\log n}logn

快速排序是一种不稳定排序：https://zhuanlan.zhihu.com/p/341201904

下面列一些这个算法模板在写的过程中需要注意的细节问题:

细节1： step3中必须要同时使用 i - 1, i 或 j , j + 1

当使用i - 1, i， x不能取左边界[l]， 当使用j, j + 1时, x不能取右边界arr[r]
否则就会出现边界问题，陷入无限递归死循环

细节2: l + (r - l) / 2和 l + (r - l + 1) / 2

C++的/是向下取整的，因此l + (r - l) / 2偏向左边界[l] =>用j , j + 1;
人为+1/2进行四舍五入，因此l + (r - l + 1) / 2偏向右边界[r] => 用 i - 1, i

细节3: while (arr[i] < x); while (arr[j] > x);就连等于边界值x时都要交换

细节4: int i = l - 1, j = r + 1;初始化时i, j在两边界外

不难发现这个算法有许多边界问题需要注意，因此还是建议背模板的!

归并排序

归并排序基于分治思想

Step1: 以中间点为分界点: mid = l + (r - l) / 2

Step2: 递归排序left, right区间;

Step3: 归并，把两个有序子数组合二为一 。⭐️

算法实现

实现难点主要在于Step3双指针实现：

  • 第一个指针最初指向第一个有序子数组(从小到大)的开头，min1min_1min1​
    第二个指针最初指向第一个有序子数组(从小到大)的开头，min2min_2min2​

  • 对两个指针指向的数循环比较，将较小的数归并回原数组，然后将其指针右移，继续比较；
    直到两个指针都指向其数组的末端，完成当轮次归并。

归并排序模板：

#include <vector>
#include <iostream>

using namespace std;

vector<int> tmp;//要开辟一个数组做归并时的临时存储

void merge_sort(vector<int> &arr, int l, int r) {
    // 递归边界
	if (l >= r) return;
    
    // step1
    int mid = l + (r - l) / 2;
    
    // step2
    merge_sort(arr, l, mid);
    merge_sort(, mid + 1, r);
    
    // step3
    int k = 0;// tmp数组中已经归并的元素数量
    int i = l, j = mid + 1;
    while (i <= mid && j <= r) {
        if (arr[i] <= arr[j]) tmp[k ++ ] = arr[i ++];
        else tmp[k ++] = arr[j ++];
    }
    //复制末尾
    while (i <= mid) tmp[k ++] = arr[i ++];
    while (j <= r) tmp[k ++] = arr[j ++];
    //复制回原数组进行覆盖
    for (int i = l, j = 0; j < k; i ++, j ++) arr[i] = tmp[j];
}

int main() {
	int n;
	cin >> n;

	vector<int>arr(n);
    tmp.resize(n);
    
	for (int i = 0; i < n; i ++) scanf("%d", &arr[i]);

	merge_sort(arr, 0, n - 1);

	for (int i = 0; i < n; i ++) printf("%d ", arr[i]);

	return 0;
}

复杂度分析

时间复杂度O(nlog2n){O(nlog_2n)}O(nlog2​n) (妥妥的) 空间复杂度O(n){O(n)}O(n)（有额外数组tmptmptmp）稳定

时间复杂度: 比较和赋值的时间复杂度都是 O(n){O(n)}O(n)，每次递归数组规模减半，即递归深度为log2n{log_2n}log2​n ，也即总共需要log2n{log_2n}log2​n次O(n){O(n)}O(n)的比较和赋值，因此总时间复杂度为O(nlog2n){O(nlog_2n)}O(nlog2​n) 。

空间复杂度: 递归深度log2n{log_2n}log2​n，即函数栈开销log2n{log_2n}log2​n，额外数组tmptmptmp为O(n){O(n)}O(n)，因此总空间复杂度为O(n){O(n)}O(n)。

不要忘记了归并排序的tmp数组的define或resize，不然会出现段错误

应用:求逆序对

使用归并排序可以在O(nlog2n){O(nlog_2n)}O(nlog2​n)，求一个长度为n的序列的逆序对的个数。

  1. 递归对左右两半排序时，可以把左右两半各自内部的逆序对数作为子问题，在合并时只需要考虑“左边一半里一个较大的数”与“右边一半里一个较小的数”构成逆序对的情形:

    res = merge_sort(arr, l, mid) + merge_sort(arr, mid + 1, r)

  2. 合并两个有序子序列arr[l∼mid]{arr[l \sim mid]}arr[l∼mid]与arr[mid+1∼r]{arr[mid + 1 \sim r]}arr[mid+1∼r]，采用两个指针i，ji，ji，j分别扫描，当出现arr[i]>arr[j]arr[i] > arr[j]arr[i]>arr[j]时，说明arr[i∼mid]{arr[i \sim mid]}arr[i∼mid]中所有元素都与arr[j]arr[j]arr[j]满足逆序对，数量为mid−i+1mid - i + 1mid−i+1。

    if(arr[i] > arr[j]) res += mid - i + 1;

#include <vector>
#include <iostream>

using namespace std;

typedef long long LL;

const int N = 5e5 + 7;

int n;
vector<int> tmp;

LL merge_sort(vector<int> &arr, int l, int r) {
    if (l >= r) return 0;

    int mid = l + (r - l) / 2;
    LL res = merge_sort(arr, l, mid) + merge_sort(arr, mid + 1, r);

    int k = 0;
    int i = l, j = mid + 1;
    while (i <= mid && j <= r) {
        if (arr[i] <= arr[j]) tmp[k ++] = arr[i ++];
        else {
            tmp[k ++] = arr[j ++];
            res += mid - i + 1; // 从i到mid全是满足的逆序对
        }
    }

    while (i <= mid) tmp[k ++] = arr[i ++];
    while (j <= r) tmp[k ++] = arr[j ++];

    for (int i = l, j = 0; j < k; i ++, j ++) arr[i] = tmp[j];
    return res;
}
